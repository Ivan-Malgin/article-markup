Замыкания в JavaScript для начинающих

Антон Иванов
Март 2026 · 8 минут чтения

Замыкания — это функция вместе с лексическим окружением, в котором она была объявлена. Они позволяют функции помнить переменные из внешней области видимости даже после того, как внешняя функция завершила выполнение.

Как работают замыкания

Когда функция создаётся, она сохраняет ссылку на своё лексическое окружение. В результате вложенная функция продолжает иметь доступ к переменным внешней функции.

// простой счётчик на замыканиях
function makeCounter() {
  let count = 0;
  return function() {
    return ++count;
  };
}
const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2
Замыкания — мощный инструмент. Их используют для инкапсуляции состояния и создания фабрик функций.
